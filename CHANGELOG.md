# Changelog

## 1.0.0 - 21 July 2026

Production readiness release.

### Added

- Visible app version and release date.
- Documentation for the storage model, backup format, CSV support, accessibility and deployment.
- Release notes and known limitations.

### Improved

- Modal keyboard handling, Escape-to-close behaviour, focus trapping and focus restoration.
- Destructive action styling for reset and recovery-point removal.
- Duplicate-click protection for export and backup actions.
- Backup metadata now records the app version.

### Verified

- Static JavaScript syntax.
- Whitespace checks.
- Responsive desktop, tablet and mobile smoke tests.
- Local browser console checks.
